org.jsimpledb.annotation

Annotation Type JTransient


  • @Retention(value=RUNTIME)
@Target(value=METHOD)
@Documented
public @interface JTransient
    Eliminates the annotated method from consideration for JSimpleDB field auto-generation.

    This annotation is ignored on methods that also have a @JField, @JSetField, @JListField, or @JMapField annotation.

    It is only useful on non-abstract methods in classes for which both JSimpleClass.autogenFields() and JSimpleClass.autogenNonAbstract() are true.

    See Also:
    JSimpleClass.autogenFields(), JSimpleClass.autogenNonAbstract()
