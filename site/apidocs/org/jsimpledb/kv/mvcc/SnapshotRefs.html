org.jsimpledb.kv.mvcc

Class SnapshotRefs

  • java.lang.Object
    • org.jsimpledb.kv.mvcc.SnapshotRefs

  • @ThreadSafe
public class SnapshotRefs
extends Object
    Holds a count of references to a CloseableKVStore and actually close()'s it when the reference count reaches zero.

    Instances are thread safe.
